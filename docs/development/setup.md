# Setup

## Stack assumption
Angular, TypeScript, RxJS, browser runtime.

## Local setup notes
Use Node LTS and the project lockfile. Backend API base URL must be configurable via Angular environment files or runtime config, not hard-coded in components.

## Baseline commands
- install: `npm ci`
- run: `npm start`

## First sanity check
After setup, verify that the application starts and that the smallest relevant test or smoke command succeeds.
